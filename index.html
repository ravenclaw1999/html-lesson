Welcome. Today, you will learn HTML.

It contains the content of our website.

This includes words, pictures, videos, links, etc.



There are also languages called CSS (Cascading Style Sheets) and JS (JavaScript).

In short, CSS is used to style our website and in general,

to make our website more organized and pretty.

And JS is used to add interactivity to our websites.

An example of this would be a button.
